ES6+

¿Qué es ES6+?

ES6+, también conocido como ECMAScript 2015+ o JavaScript ES6+, se refiere a las versiones de ECMAScript que comenzaron con la versión 6 (ES6) lanzada en 2015 y las actualizaciones posteriores (ES7, ES8, etc.).

ECMAScript es el estándar sobre el cual se basa JavaScript, y ES6 fue una de las actualizaciones más significativas que introdujo muchas características nuevas que mejoraron la sintaxis y funcionalidad del lenguaje.

En este módulo veremos diferentes características nuevas que nos facilitarán el día a día con nuestro código.

Destructuring

El destructuring es una característica de ES6 que permite extraer valores de arrays u objetos y asignarlos a variables de una manera más concisa y legible.

Esta técnica simplifica la manipulación de estructuras complejas de datos.

Vamos a desglosarlos con diferentes ejemplos para ver todo lo que abarca su utilidad.

Destructuring de arrays

La desestructuración de arrays permite extraer elementos individuales de un array y asignarlos a variables en una sola línea.


            const fruits = ['apple', 'banana', 'cherry'];

            // Asignación tradicional
            const firstFruit = fruits[0];
            const secondFruit = fruits[1];

            // Usando desestructuración
            const [first, second, third] = fruits;

            console.log(first);  // Output: 'apple'
            console.log(second); // Output: 'banana'
            console.log(third);  // Output: 'cherry'

En este ejemplo, hemos extraído los valores apple, banana y cherry del array fruits y los hemos asignado a las variables first, second y third.

Destructuring de objetos

La desestructuración de objetos permite extraer propiedades de un objeto y asignarlas a variables.


            const person = {
                name: 'John Doe',
                age: 30, 
                profession: 'Developer'
            };

            // Asignación tradicional
            const personName = person.name;
            const personAge = person.age;

            // Usando desestructuración
            const { name, age, profession } = person;

            console.log(name);       // Output: 'John Doe'
            console.log(age);        // Output: 30
            console.log(profession); // Output: 'Developer'

Aquí, hemos extraído las propiedades name, age y profession del objeto person y las hemos asignado a variables con los mismos nombres.

Valores por Defecto

Puedes proporcionar valores por defecto en caso de que la propiedad no exista en el objeto o el array.


            const person = {
                name: 'Jane Doe'
            };

            // Usando desestructuración con valores por defecto
            const { name, age = 25 } = person;

            console.log(name); // Output: 'Jane Doe'
            console.log(age);  // Output: 25 (valor por defecto)

En este caso, como age no está definido en el objeto person, se usa el valor por defecto 25.

Destructurin Anidado

Puedes usar desestructuración anidada para extraer valores de estructuras de datos más complejas.


            const user = {
                id: 101,
                info: {
                    name: 'Alice',
                    location: {
                    city: 'Wonderland',
                    country: 'Fantasy'
                    }
                }
            };

            // Usando desestructuración anidada
            const { info: { name, location: { city, country } } } = user;

            console.log(name);    // Output: 'Alice'
            console.log(city);    // Output: 'Wonderland'
            console.log(country); // Output: 'Fantasy'

En este ejemplo, hemos extraído name, city y country de un objeto anidado.

Intercambio de Variables

La desestructuración también puede ser usada para intercambiar valores entre dos variables de manera concisa.


            let a = 1;
            let b = 2;

            // Usando desestructuración
            [a, b] = [b, a];

            console.log(a); // Output: 2
            console.log(b); // Output: 1

Spread Operator

El spread operator permite expandir elementos de un array o propiedades de un objeto en lugares donde se esperan múltiples elementos.

Esto es útil para combinar arrays, copiar arrays, y pasar elementos de un array como argumentos a funciones.

Ejemplos con Arrays

Combinar arrays:


            const avengers = ['Iron Man', 'Thor', 'Hulk'];
            const guardians = ['Star-Lord', 'Gamora', 'Drax'];

            const combinedTeam = [...avengers, ...guardians];
            console.log(combinedTeam); // Output: ['Iron Man', 'Thor', 'Hulk', 'Star-Lord', 'Gamora', 'Drax']

Copiar arrays:


            const originalTeam = ['Batman', 'Superman', 'Wonder Woman'];
            const copiedTeam = [...originalTeam];

            console.log(copiedTeam); // Output: ['Batman', 'Superman', 'Wonder Woman']

Pasar elementos como argumentos:


            const jedi = ['Luke', 'Yoda', 'Obi-Wan'];

            function greetJedi(master1, master2, master3) {
                return `Greetings ${master1}, ${master2}, and ${master3}`;
            }

            console.log(greetJedi(...jedi)); // Output: Greetings Luke, Yoda, and Obi-Wan

Ejemplos con Objetos

Combinar objetos:


            const batman = { name: 'Bruce Wayne', alias: 'Batman' };
            const superman = { name: 'Clark Kent', alias: 'Superman' };

            const combinedHeroes = { ...batman, ...superman };
            console.log(combinedHeroes); // Output: { name: 'Clark Kent', alias: 'Superman' }

Copiar objetos:


            const originalHero = { name: 'Peter Parker', alias: 'Spider-Man' };
            const copiedHero = { ...originalHero };

            console.log(copiedHero); // Output: { name: 'Peter Parker', alias: 'Spider-Man' }

Actualizar propiedades:


            const ironMan = { name: 'Tony Stark', suit: 'Mark I' };
            const updatedIronMan = { ...ironMan, suit: 'Mark L' };

            console.log(updatedIronMan); // Output: { name: 'Tony Stark', suit: 'Mark L' }

Rest Parameter

El rest parameter permite agrupar el número indefinido de argumentos en un array.

Se usa principalmente en la definición de funciones para capturar todos los argumentos restantes.

Ejemplo con Funciones

Función con parámetros rest:


            const assembleAvengers = (...heroes) => {
                return heroes.join(', ');
            }

            console.log(assembleAvengers('Iron Man', 'Thor', 'Hulk', 'Black Widow')); // Output: Iron Man, Thor, Hulk, Black Widow

Parámetros rest en funciones:


            const teamUp = (leader, ...members) => {
                console.log(`Leader: ${leader}`);
                console.log(`Members: ${members.join(', ')}`);
            }

            teamUp('Professor X', 'Cyclops', 'Wolverine', 'Storm');
            // Output:
            // Leader: Professor X
            // Members: Cyclops, Wolverine, Storm

Uso en Objetos

Extraer Propiedades Rest en Objetos:


            const mutant = { name: 'Logan', alias: 'Wolverine', team: 'X-Men' };
            const { alias, ...rest } = mutant;

            console.log(alias); // Output: 'Wolverine'
            console.log(rest); // Output: { name: 'Logan', team: 'X-Men' }

Casos de Uso Combinados

Uso Combinado en Funciones y Arrays:


            const droids = ['R2-D2', 'C-3PO', 'BB-8'];

            function repairDroids(repairman, ...droids) {
                return `Repairman ${repairman} will fix: ${droids.join(', ')}`;
            }

            console.log(repairDroids('Anakin', ...droids)); // Output: Repairman Anakin will fix: R2-D2, C-3PO, BB-8

Actualizar Arrays Inmutables:


            const originalAvengers = ['Iron Man', 'Thor', 'Hulk'];

            const updatedAvengers = [...originalAvengers, 'Captain America', 'Black Widow'];
            console.log(updatedAvengers); // Output: ['Iron Man', 'Thor', 'Hulk', 'Captain America', 'Black Widow']

Filtrar Propiedades de un Objeto:


            const wizard = { name: 'Harry', house: 'Gryffindor', position: 'Seeker' };

            const { position, ...rest } = wizard;
            console.log(rest); // Output: { name: 'Harry', house: 'Gryffindor' }

El spread operator facilita la combinación y copia de arrays y objetos, mientras que el rest parameter permite manejar argumentos variables en funciones y extraer propiedades restantes de objetos.

Modules

Los módulos en ES6+ permiten dividir el código en archivos reutilizables y organizados. Esta característica mejora la mantenibilidad y la claridad del código, permitiendo exportar e importar funcionalidades entre diferentes archivos.

ES6 introduce dos palabras clave principales para trabajar con módulos: export e import.

Exportación

Existen dos tipos principales de exportaciones en ES6: exportaciones nombradas (named exports) y exportaciones por defecto (default exports).

Exportaciones Nombradas

Las exportaciones nombradas permiten exportar múltiples valores desde un módulo. Cada valor exportado debe ser importado con el mismo nombre.


            // file: superheroes.js
            export const hero = 'Spider-Man';
            export const villain = 'Green Goblin';

            export function saveTheDay() {
                console.log(`${hero} saves the day!`);
            }

Exportaciones por Defecto

Las exportaciones por defecto permiten exportar un solo valor o una función desde un módulo. Este valor puede ser importado con cualquier nombre.


            // file: superpower.js
            const superpower = 'Web-Slinging';
            export default superpower;

Importación

Las importaciones permiten traer funciones, variables, u objetos de otros módulos al contexto actual.

Importaciones Nombradas

Las importaciones nombradas requieren el nombre exacto de lo que se está importando desde el módulo.


            // file: main.js
            import { hero, villain, saveTheDay } from './superheroes.js';

            console.log(hero); // Output: 'Spider-Man'
            console.log(villain); // Output: 'Green Goblin'
            saveTheDay(); // Output: 'Spider-Man saves the day!'

Importaciones por Defecto

Las importaciones por defecto pueden ser nombradas como desees al importarlas.


            // file: main.js
            import superpower from './superpower.js';

            console.log(superpower); // Output: 'Web-Slinging'

Reexportaciones

Un módulo puede reexportar cosas que ha importado de otros módulos, facilitando la creación de puntos de entrada unificados.


            // file: allHeroes.js
            export { hero, villain, saveTheDay } from './superheroes.js';
            export { default as superpower } from './superpower.js';

            // file: main.js
            import { hero, villain, saveTheDay, superpower } from './allHeroes.js';

            console.log(hero); // Output: 'Spider-Man'
            console.log(villain); // Output: 'Green Goblin'
            saveTheDay(); // Output: 'Spider-Man saves the day!'
            console.log(superpower); // Output: 'Web-Slinging'

Ejemplo Completo

Vamos a construir un ejemplo completo que utiliza módulos para gestionar un equipo de superhéroes y sus habilidades.


            // file: characters.js
            export const heroes = ['Batman', 'Wonder Woman', 'Flash'];
            export const villains = ['Joker', 'Lex Luthor', 'Reverse-Flash'];
        

            // file: team.js
            export function assembleTeam(teamName, members) {
                return `Team ${teamName} consists of: ${members.join(', ')}`;
            }

            const mainTeam = ['Superman', 'Batman', 'Wonder Woman'];
            export default mainTeam;
        

            // file: main.js
            import mainTeam, { assembleTeam } from './team.js';
            import { heroes, villains } from './characters.js';

            console.log(assembleTeam('Justice League', mainTeam));
            // Output: Team Justice League consists of: Superman, Batman, Wonder Woman

            console.log(`Heroes: ${heroes.join(', ')}`);
            // Output: Heroes: Batman, Wonder Woman, Flash

            console.log(`Villains: ${villains.join(', ')}`);
            // Output: Villains: Joker, Lex Luthor, Reverse-Flash

Métodos de arrays ES6+

ES6+ introdujo varias mejoras en los métodos de arrays, haciendo que el manejo de colecciones de datos sea más fácil y más eficiente.

Estos métodos ayudan a realizar operaciones comunes de manera más concisa y legible.

Veamos qué utilidades podemos darles a cada uno de estos nuevos métodos:

map()

El método map crea un nuevo array con los resultados de aplicar una función a cada uno de los elementos del array original.

Veamos varios ejemplos:

Convertir edades en años perros (1 año humano = 7 años perros)


            const ages = [2, 5, 8];
            const dogAges = ages.map(age => age * 7);
            console.log(dogAges); // Output: [14, 35, 56]

Generar un nuevo array de objetos con nombre y longitud


            const names = ['Alice', 'Bob', 'Charlie'];
            const nameLengths = names.map(name => ({ name, length: name.length }));
            console.log(nameLengths); // Output: [{name: 'Alice', length: 5}, {name: 'Bob', length: 3}, {name: 'Charlie', length: 7}]

Convertir un array de precios en euros a dólares (1 euro = 1.2 dólares)


            const pricesInEuros = [10, 20, 30];
            const pricesInDollars = pricesInEuros.map(price => price * 1.2);
            console.log(pricesInDollars); // Output: [12, 24, 36]

filter()

El método filter crea un nuevo array con todos los elementos que cumplan una condición definida en una función.

Veamos varios ejemplos:

Filtrar números pares


            const numbers = [1, 2, 3, 4, 5, 6];
            const evenNumbers = numbers.filter(number => number % 2 === 0);
            console.log(evenNumbers); // Output: [2, 4, 6]

Filtrar nombres que empiezan con 'A’


            const names = ['Alice', 'Bob', 'Alex', 'Charlie'];
            const namesStartingWithA = names.filter(name => name.startsWith('A'));
            console.log(namesStartingWithA); // Output: ['Alice', 'Alex']

Filtrar edades mayores o iguales a 18


            const ages = [12, 18, 22, 15, 30];
            const adults = ages.filter(age => age >= 18);
            console.log(adults); // Output: [18, 22, 30]

reduce()

El método reduce aplica una función a un acumulador y a cada elemento del array (de izquierda a derecha) para reducirlo a un único valor.

Veamos varios ejemplos:

Sumar todos los números del array


            const numbers = [1, 2, 3, 4, 5];
            const sum = numbers.reduce((acc, curr) => acc + curr, 0);
            console.log(sum); // Output: 15

Agrupar objetos por una propiedad


            const people = [
                { name: 'Alice', age: 21 },
                { name: 'Bob', age: 25 },
                { name: 'Charlie', age: 21 }
            ];
            const groupedByAge = people.reduce((acc, person) => {
                if (!acc[person.age]) acc[person.age] = [];
                acc[person.age].push(person);
                return acc;
            }, {});
            console.log(groupedByAge);
            // Output: {
            //   '21': [{ name: 'Alice', age: 21 }, { name: 'Charlie', age: 21 }],
            //   '25': [{ name: 'Bob', age: 25 }]
            // }

find()

El método find devuelve el primer elemento del array que cumpla con una condición especificada en una función.

Si ningún elemento cumple con la condición, devuelve undefined.

Veamos varios ejemplos:

Encontrar el primer número mayor que 3


            const numbers = [1, 2, 3, 4, 5];
            const firstGreaterThanThree = numbers.find(number => number > 3);
            console.log(firstGreaterThanThree); // Output: 4

Encontrar el primer nombre con más de 5 caracteres


            const names = ['Bob', 'Charlie', 'Alice'];
            const longName = names.find(name => name.length > 5);
            console.log(longName); // Output: 'Charlie'

Encontrar el primer objeto con una propiedad específica


            const people = [
                { name: 'Alice', age: 21 },
                { name: 'Bob', age: 25 },
                { name: 'Charlie', age: 21 }
            ];
            const personNamedBob = people.find(person => person.name === 'Bob');
            console.log(personNamedBob); // Output: { name: 'Bob', age: 25 }

every()

El método every comprueba si todos los elementos del array cumplen con una condición especificada en una función.

Devuelve true si todos los elementos cumplen con la condición, de lo contrario, devuelve false.

Veamos varios ejemplos:

Comprobar si todos los números son positivos


            const numbers = [1, 2, 3, 4, 5];
            const allPositive = numbers.every(number => number > 0);
            console.log(allPositive); // Output: true

Comprobar si todos los nombres tienen más de 3 caracteres


            const names = ['Alice', 'Bob', 'Charlie'];
            const allLongNames = names.every(name => name.length > 3);
            console.log(allLongNames); // Output: false

Comprobar si todos los objetos tienen una propiedad específica


            const people = [
                { name: 'Alice', age: 21 },
                { name: 'Bob', age: 25 },
                { name: 'Charlie', age: 21 }
            ];
            const allHaveAge = people.every(person => person.age !== undefined);
            console.log(allHaveAge); // Output: true

some()

El método some comprueba si al menos un elemento del array cumple con una condición especificada en una función.

Devuelve true si al menos un elemento cumple con la condición, de lo contrario, devuelve false.

Veamos varios ejemplos:

Comprobar si al menos un número es mayor que 4


            const numbers = [1, 2, 3, 4, 5];
            const someGreaterThanFour = numbers.some(number => number > 4);
            console.log(someGreaterThanFour); // Output: true

Comprobar si al menos un nombre tiene más de 5 caracteres


            const names = ['Bob', 'Charlie', 'Alice'];
            const someLongNames = names.some(name => name.length > 5);
            console.log(someLongNames); // Output: true

includes()

El método includes determina si un array incluye un determinado elemento, devolviendo true o false según corresponda.

Este método ya lo hemos visto anteriormente, pero nos servirá combinado con los que estamos aprendiendo ahora mismo.

Veamos varios ejemplos:

Comprobar si el array incluye el número 3


            const numbers = [1, 2, 3, 4, 5];
            const includesThree = numbers.includes(3);
            console.log(includesThree); // Output: true

Comprobar si el array incluye un objeto específico (requiere referencia exacta)


            const alice = { name: 'Alice', age: 21 };
            const bob = { name: 'Bob', age: 25 };
            const people = [alice, bob];
            const includesAlice = people.includes(alice);
            console.log(includesAlice); // Output: true
