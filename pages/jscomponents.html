Componentes JavaScript

Los componentes son un grupo de archivos, generalmente JavaScript y CSS, que conformarán un elemento o conjunto de elementos con la lógica separada del resto del proyecto. Esto nos permitirá encapsular en diferentes carpetas de nuestro proyecto diferentes funcionalidades e interfaces de usuario.

Esto nos resultará muy útil por las diferentes ventajas que esto conlleva, sobretodo para proyectos más grandes.

Ventajas

  • Reutilización: Los componentes se diseñan para ser reutilizables en diferentes partes de una aplicación o incluso en proyectos diferentes.
  • Mantenimiento: Al separar la funcionalidad en componentes individuales, se facilita el mantenimiento y la escalabilidad de la aplicación.

Desventajas

  • Curva de Aprendizaje: Puede haber una curva de aprendizaje inicial compleja al familiarizarse con los conceptos de componentes.
  • Complejidad: En proyectos pequeños, la introducción de componentes puede parecer innecesaria y agregar complejidad.

Este enfoque es uno entre muchos, por lo que se pueden diseñar los componentes o la lógica interna de diferentes maneras siempre y cuando el código sea legible y funcional.

Ejemplo

Un ejemplo visual nos permitirá entender mucho mejor el concepto. Para ello, vamos a crear un proyecto directamente con VITE, puesto que para importar archivos de css directamente en nuestro js necesitaremos hacer ciertas configuraciones en el proyecto. Al ser modular, VITE ya viene preparado para ello.

Estructura

Lo primero que haremos será limpiar los contenidos que nos trae por defecto VITE al crear un proyecto.

Una vez hecho, crearemos la siguiente estructura de carpetas, donde en src hemos creado la carpeta components y dentro nuestro nuevo componente OurButton.

De momento solo creamos carpetas y archivos, de momento vacíos, para conseguir una estructura como la aportada.

Recordad ejecutar el comando npm i en la terminal para instalar las dependencias de VITE y poder ejecutar el proyecto.

index.html

En este archivo no tocaremos nada (en este ejemplo), pero recordad que hay un div con el atributo id = "app".


            <!doctype html>
            <html lang="en">
                <head> 
                    <meta charset="UTF-8" />
                    <link rel="icon" type="image/svg+xml" href="/vite.svg" />
                    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
                    <title>Vite App</title>
                </head>
                <body>
                    <div id="app"></div>
                    <script type="module" src="/main.js"></script>
                </body>
            </html>

OurButton.js

El contenido de nuestro componente será esta pequeña función, que recibirá un texto y un color de fondo, aunque puede ser lo que necesitéis. Además, importaremos nuestro css del componente para poder implementar ahí los estilos del componente y que se rendericen correctamente.


            <script type="module">
            import "./OurButton.css";

            export const ourButton = (texto, bgColor) => {
                return `<button style="background-color: ${bgColor}" class="ourButton">${texto}</button>`;
            };
            </script>

OurButton.css

El CSS de nuestro botón será algo muy sencillo, os proponemos lo siguiente como ejemplo:


            .ourButton {
                border-radius: 8px;
                border: none;
                background-color: #ffd17b;
                padding: 5px 10px;
            }

main.js

Ahora ya podremos utilizar nuestro componente ourButton tantas veces como queramos y donde queramos. Por ejemplo, importamos la función, seleccionamos el div con el id app del index.html, y le añadimos contenido de HTML directamente.

Cada vez que queramos meter un nuevo botón lo haremos con el uso de la función creada previamente, que nos retornará un string.

La interpolación con ${} será tu aliada en estos casos.


            <script type="module">
            import { ourButton } from './src/components/OurButton/OurButton';
            import './style.css';

            const app = document.querySelector("#app");

            app.innerHTML = `
                ${ourButton("Este es el texto elegido para el botón", "#9a9aff")}
                ${ourButton("Login", "#ffaaaa")}
                <ol>
                    <li>Otros</li>
                    <li>Contenidos</li>
                </ol>
                ${ourButton("Otro texto!")}
            `;
            </script>
