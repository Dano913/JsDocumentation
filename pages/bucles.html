Bucles

Los bucles son una herramienta muy útil en JavaScript y en cualquier otro lenguaje de programación.

Sirven para ejecutar una acción o conjunto de acciones varias veces, según una condición o una serie de condiciones.

Bucle while

El bucle while es un tipo de bucle muy común en JavaScript. Se utiliza para ejecutar un bloque de código mientras se cumpla una determinada condición.

La sintaxis de un bucle while es la siguiente:


            while (condición) {
                // código a ejecutar
            }

Aquí tienes un ejemplo de un bucle while que imprime los números del 2 al 11:

Es muy importante que la condición tenga un punto de ruptura, es decir que en algún momento el bucle while se rompa, puesto que si no se crearía un bucle infinito.


            let i = 1

            while (i <= 10) {
                i++;
                console.log(i);
            }

Concepto de un bucle

Teniendo como referencia el ejemplo anterior del while, explicaremos de manera más visual lo que es un bucle y como este funciona.

No todos nuestros bucles serán iguales, algunos irán restando, otros sumando, podemos hacer que vayan multiplicando o lo que se nos ocurra, es algo muy modificable.

Sin embargo casi todos funcionarán de la misma manera, tendrán un inicio, una iteración (modificación de ese inicio) y una condición que en algún momento deje de cumplirse.

Inicio de un bucle

Aquí podemos ver de manera un poco más visual cómo se inicia nuestro bucle, en este punto sólo nos interesa ver la variable inicial creada, en este caso nuestra variable se llama "i" y comienza con el valor numérico "0".

Iteración sobre la variable inicial

Lo importante es cómo al final de la acción o acciones que realice siempre modificará la variable inicial, en este caso le está sumando 1 a lo que valía, es decir, ahora la variable "i" tiene el valor "1" y así sucesivamente después de cada vuelta que de nuestro bucle.

¿Se sigue cumpliendo la condición?

La condición puede ser la que queramos, en este caso es si la "i" sigue siendo menor de 10 o no, como en este caso la "i" vale 11 ya, NO se cumple la condición y por lo tanto NO continúa la ejecución del bucle, las veces que se compruebe y SI cumpla la condición, seguirá con la ejecución del bucle.

Final de concepto

Todos estos pasos se realizan de manera "interna" por el bucle en CADA UNA de las vueltas que este mismo realice, obviamente podemos elegir nosotros:

El valor inicial de nuestra variable.

La manera de iterar esa variable en cada vuelta del bucle, sumar 1, restar 3, multiplicar, etc... (lo más normal es sumar de 1 en 1).

La condición que se debe cumplir en cada vuelta del bucle para continuar.

Por lo tanto al ser tan libre, todos los ejemplos que pongamos se quedarán cortos, hay que utilizar esto una y otra vez. Os invitamos a jugar con esto libremente.

Bucle Do While

El bucle do while será muy similar al bucle while visto anteriormente, pero este hará algo antes de comprobar, de ahí su nombre, aquí os dejamos un ejemplo y de nuevo comprobad en vuestro propio código cómo se comporta:


            let i = 1;

            do {
            i++;
            console.log(i);
            } while (i <= 10);

Bucle for

Es el bucle más básico y funcional.

Se inicia con la palabra reservada for y a continuación la condición de iteración.

Es decir, entre paréntesis lleva la inicialización let index = 0, la condición de ruptura index < 5 y el incremento index ++.

Como podemos ver es muy parecido al bucle while y por supuesto la explicación de concepto se mantiene. Simplemente se escribe de otra manera más customizable, en este caso con la siguiente sintaxis:


            for (let index = 0; index < 5; index++) {
                // código que se repite 5 veces en total
            }

Bucle for para recorrer tipos de datos iterables, como los arrays

Comúnmente utilizamos bucles para acceder a cada posición de un dato iterable, como vimos anteriormente a los elementos de un array podemos acceder mediante su posición en el array poniendo los [posicion] por lo tanto...

¿Por qué no accedemos a estos elementos con la ayuda de un bucle para pasar por todos ahorrándonos código?

Acceder a los elementos del array SIN bucles de uno en uno

Como podemos ver, esta tarea sería muy tediosa, y que deciros si tuviéramos 1000 elementos dentro del array…


            let frutas = ["pomelo", "pera", "manzana", "naranja", "mandarina"];

            // sin bucles...
            console.log(frutas[0]); // pomelo
            console.log(frutas[1]); // pera
            console.log(frutas[2]); // manzana
            console.log(frutas[3]); // naranja
            console.log(frutas[4]); // mandarina

Acceder a los elementos del array con el bucle for de uno en uno

Como podemos ver en nuestro bucle for nos aprovechamos de esa variable "i" que hemos creado y vamos de 0 hasta el final del array y la utilizamos entre corchetes al lado del array para ir accediendo a cada elemento en cada vuelta del bucle:


            let frutas = ["pomelo", "pera", "manzana", "naranja", "mandarina"];

            // con bucle for...
            // empezamos en 0;
            // siempre y cuando la i sea menor que la longitud del array frutas seguimos
            // sumamos 1 a la i en cada vuelta del bucle
            for (let i = 0; i < frutas.length; i++) {
                console.log(frutas[i]); // pomelo // pera // manzana // naranja // mandarina
            }

Bucles anidados

Podemos anidar tantos bucles como queramos, utilizar bloques condicionales dentro y utilizar cualquiera de las anteriores herramientas vistas para crear una solución a un problema, por ejemplo: tenemos un array de frutas y queremos contar cuantas letras tiene cada una, (hay soluciones más fáciles para este "problema") pero en este caso utilizamos dos bucles, uno para ir cogiendo cada elemento del array y otro para ir contando cada letra del string:


            let frutas = ["pomelo", "pera", "manzana", "naranja", "mandarina"];

            let frutasLength = [];

            for (let i = 0; i < frutas.length; i++) {
                let fruta = frutas[i];
                for (let j = 0; j <= fruta.length; j++) {
                    frutasLength[i] = j
                }
            }

            console.log(frutasLength); // [6, 4, 7, 7, 9]

Entendiendo a fondo el ejemplo anterior

Como podemos ver en el ejemplo anterior estamos utilizando dos bucles anidados para resolver el problema:

  • Creamos una variable porque necesitamos guardar las longitudes de cada fruta, en este caso la variable la iniciamos como array vacío.
  • Recorremos con un bucle for el array para mirar cada fruta de una en una.
  • Recorremos cada string para poder acceder a cada una de las letras de una en una.
  • Introducimos una nueva longitud de fruta en el array en la última posición disponible.

Derivaciones del bucle for

El bucle for es el predecesor de los distintos bucles que veremos a continuación, estos bucles servirán para lo mismo, solo que la sintaxis varía, pero básicamente son lo mismo.

Bucle for... of

El bucle for...of se comportará como un bucle for, pero tiene un par de limitaciones en cuanto a hacer bucles muy personalizados o cambiar el valor del índice, puesto que no se puede.

Es un tipo de bucle creado exclusivamente para recorrer strings o arrays al completo de manera más abreviada que el bucle for normal.

Comparación con el bucle for

Recorrer un array con un bucle for se vería como algo así:


            const numbers = [1, 2, 3, 4, 5, 6, 7];

            //           usamos numbers.length
            for (let i = 0; i < numbers.length; i++) {

                /*
                creamos la variable element que guarde el valor de numbers[i]
                (cada elemento del array en cada vuelta del mismo)
                simplemente por comdidad a la hora de continuar con el código
                */
                const element = numbers[i];

                console.log(element);

            }

Con el bucle for...of, nos resultará más cómodo

Como podemos ver a continuación, para recorrer un array o string, será mucho más asequible, y por dentro hará los mismos pasos pero nos los está ocultando para que no sea tan "difícil" recorrer un array con un bucle for.


            const numbers = [1, 2, 3, 4, 5, 6, 7];

            // creamos la variable element
            // por dentro actuará de la misma manera que en el bucle for
            // obteniendo el valor de numbers[i]
            // usamos después del of el array numbers
            // que es el que queremos recorrer
            // el bucle forof cogerá la longitud de numbers
            // y numbers[i] para guardarlo en element
            for (const element of numbers) {
                console.log(element);
            }

Bucle for in

En bucle for in está enfocado a "recorrer" objetos.

Un objeto como tal no se puede recorrer a no ser que tenga ciertas propiedades que lo conviertan en array.

Pero el bucle for in nos da un "life hack" para poder simular recorrer un objeto en JavaScript.

Bucle for para recorrer un objeto


            const user = {
                email: "user@user.com",
                password: "odsfofjowf"
            }

            // Object.keys me devolverá un array con todas las claves
            // del objeto que le mande
            const allTheKeysOfMyObject = Object.keys(user);
            // allTheKeysOfMyObject = ["email", "password"];

            // recorremos ese array de claves con el bucle
            for (let i = 0; i < allTheKeysOfMyObject.length; i++) {

                // nos guardamos cada clave en cada vuelta
                const key = allTheKeysOfMyObject[i];

                // en vez de usar . para acceder a la propiedad de un objeto
                // usamos los corchetes para poder obtener una propiedad
                // de nuestro objeto con la variable key
                // que en cada vuelta del bucle será una clave del objeto
                console.log(user[key]);

            }

Ahora con el bucle for in

El bucle for in llega para salvarnos un poco los platos en este caso, veamos como quedaría:


            const user = {
                email: "user@user.com",
                password: "odsfofjowf"
            }

            for (const key in user) {
                console.log(user[key]);
            }

Bucle for each

El bucle for each llega en ES6 para dar una nueva sintaxis usando callbacks y arrow functions, pero será muy similar al bucle forof.

Sólo tendremos que acostumbrarnos a la sintaxis, aunque este sólo nos servirá para recorrer arrays, puesto que es un método de array:


            const numbers = [1, 2, 3, 4, 5, 6, 7];

            numbers.forEach((number) => {
                console.log(number);
            })

Este bucle puede que al principio os cueste un poco, y aunque no es 100% necesario está bien que sepamos utilizarlo, para ello, os aportaremos un video en el que se verá mejor este concepto.
