Arrays

Un array o matriz, no es un tipo de dato como tal en JavaScript, sin embargo, nosotros lo interpretaremos como tal de momento, puesto que será un elemento que utilizaremos de manera muy asidua. Un array SIEMPRE se escribe de la misma manera.

Será una lista de elementos, irá envuelta con los corchetes [] y los elementos irán separados por comas.

Un array puede tener cualquier tipo de dato en su interior, otros arrays, objetos, números, strings, booleans...

Además, no tienen por qué ser todos los elementos que tengamos en una lista del mismo tipo, pero será lo más común. Es como si tenemos una lista de la compra y entre toda esa lista de repente nos encontramos un elemento que no debería ir en la lista, por ejemplo no sería lógico que una lista de la compra aparezca la palabra "Europa".

Lo más normal es que los arrays tengan sentido lógico.

Un array es iterable

Como vimos con los strings (cadenas de texto), que un dato sea iterable significa que sus elementos internos se clasifican por posiciones, de la 0 hasta el último elemento de 1 en 1, y por lo tanto con esa cualidad podemos utilizar bucles para "iterar" o "recorrer" el propio dato, y por lo tanto, podremos usar montones de métodos creados para hacer esto de manera más rápida o sencilla.

Posiciones en un array

Los tipos de datos que haya dentro de CADA posición de nuestro array, dan completamente igual, cada elemento estará separado por comas, y cada elemento corresponderá a una posición.


            const strings = ["hola", "mundo", "!"];
            //posiciones ->    0        1      2

Accediendo a una posición de un array

Acceder a una posición de un array se realiza mediante el uso de corchetes a continuación del nombre de la variable array y utilizando los números de cada posición.

Sin embargo, también es posible acceder a las posiciones mediante expresiones matemáticas, lógicas o de comparación.


            const array = ["elemento1", "elemento2", "elemento3"];
            // posiciones       0            1            2

            const posicion = 1;

            console.log(array[0]); // elemento1

            console.log(array[posicion]); // elemento2
                                // 1

            console.log(array[posicion + 1]); // elemento3
                            // 1 + 1 = 2

Como vimos en el ejemplo anterior, dentro de los corchetes al final siempre estamos poniendo un número, aunque estemos usando palabras o expresiones.

Estos números pueden ser:

  • Cálculos matemáticos: array[2 + 3]
  • Sentencias lógicas: array[edad > 18]
  • Cualquier otra sentencia que devuelva un número: array[funcionQueDevuelveUnNumero()]

Lo importante es que el resultado final sea una posición válida (número) para nuestro array.

Usar el índice de un bucle para acceder a CADA posición del array

Aquí se observa un bucle que recorre un array desde la posición 0 hasta la última.

El valor del índice del bucle ("i") se utiliza para acceder a cada uno de los elementos del array. La variable "i" actúa como la posición del array dentro de los corchetes.

No os preocupéis, lo veremos más adelante.


            const array = ["elemento1", "elemento2", "elemento3"];
            // posiciones       0            1            2

            for (let i = 0; i < array.length; i++) {

                console.log(array[i]); // elemento1   elemento2   elemento3

            }

Métodos de arrays

Al igual que los Strings, los arrays poseen una gran cantidad de métodos que nos simplifican el trabajo (aunque no realizan tareas mágicas, ya que todo lo que hacen se puede lograr manualmente).


            let array = [1, 2, 3, 4, 5];

            // 1. push(): Añade uno o más elementos al final del array y devuelve la nueva longitud del array.
            array.push(6);

            // 2. pop(): Elimina el último elemento de un array y lo devuelve.
            let ultimoElemento = array.pop();

            // 3. shift(): Elimina el primer elemento de un array y lo devuelve.
            let primerElemento = array.shift();

            // 4. unshift(): Añade uno o más elementos al inicio de un array y devuelve la nueva longitud del array.
            array.unshift(0);

            // 5. slice(): Devuelve una copia de una parte del array dentro de un nuevo array.
            let subArray = array.slice(2, 4);

            // 6. splice(): Cambia el contenido de un array eliminando elementos existentes y/o agregando nuevos elementos.
            array.splice(2, 1, 'a', 'b', 'c');

            // 7. concat(): Retorna un nuevo array que resulta de la concatenación de dos o más arrays.
            let array2 = ['x', 'y', 'z'];
            let newArray = array.concat(array2);

            // 8. indexOf(): Devuelve el primer índice en el que se puede encontrar un elemento dado en el array, o -1 si no está presente.
            let indice = array.indexOf(3);

            // 9. forEach(): Ejecuta una función dada una vez por cada elemento del array.
            array.forEach((elemento) => {
            console.log(elemento);
            });

            // 10. map(): Crea un nuevo array con los resultados de llamar a una función dada en cada elemento del array.
            let nuevoArray = array.map((elemento) => elemento * 2);

            // 11. filter(): Crea un nuevo array con todos los elementos que cumplan una condición dada.
            let arrayFiltrado = array.filter((elemento) => elemento > 3);

            // 12. reduce(): Aplica una función a un acumulador y a cada elemento (de izquierda a derecha) para reducirlo a un solo valor.
            let suma = array.reduce((acumulador, elemento) => acumulador + elemento, 0);

            // 13. find(): Devuelve el valor del primer elemento del array que cumple la función de prueba proporcionada.
            let encontrado = array.find((elemento) => elemento > 3);

            // 14. sort(): Ordena los elementos de un array localmente y devuelve el array ordenado.
            array.sort();

            // 15. reverse(): Invierte el orden de los elementos de un array in-place.
            array.reverse();

No es necesario memorizar todos los métodos de los arrays, ni sus funciones específicas o cómo utilizarlos.

Sin embargo, algunos de ellos son bastante útiles en el desarrollo actual y los usaremos con frecuencia en ejercicios posteriores.
