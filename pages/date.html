Date

El objeto Date en JavaScript nos permite trabajar con fechas y horas.

Es una herramienta fundamental para el desarrollo web, ya que nos permite obtener, formatear y manipular fechas para diversas aplicaciones.

Uso básico

Para crear un nuevo objeto Date, podemos utilizar la siguiente sintaxis:


            const fechaActual = new Date();

Esto creará un objeto Date que representa la fecha y hora actual. También podemos especificar una fecha y hora concreta utilizando los siguientes parámetros:


            const fechaEspecifica = new Date(año, mes, dia, horas, minutos, segundos, milisegundos);

Ejemplos de código

Obtener la fecha actual:


            const fechaActual = new Date();
            console.log(fechaActual); // Muestra la fecha y hora actual

Obtener el año actual:


            const fechaActual = new Date();
            const añoActual = fechaActual.getFullYear();
            console.log(añoActual); // Muestra el año actual

Obtener el mes actual:


            const fechaActual = new Date();
            const mesActual = fechaActual.getMonth() + 1; // Los meses en JavaScript van de 0 a 11
            console.log(mesActual); // Muestra el mes actual

Obtener el día actual:


            const fechaActual = new Date();
            const diaActual = fechaActual.getDate();
            console.log(diaActual); // Muestra el día actual

Obtener la hora actual:


            const fechaActual = new Date();
            const horaActual = fechaActual.getHours();
            console.log(horaActual); // Muestra la hora actual

Obtener los segundos actuales:


            const fechaActual = new Date();
            const segundosActuales = fechaActual.getSeconds();
            console.log(segundosActuales); // Muestra los segundos actuales

Obtener las milisegundos actuales:


            const fechaActual = new Date();
            const milisegundosActuales = fechaActual.getMilliseconds();
            console.log(milisegundosActuales); // Muestra las milisegundos actuales

Formatear una fecha:


            const fechaActual = new Date();
            const fechaFormateada = fechaActual.toLocaleDateString('es-ES');
            console.log(fechaFormateada); // Muestra la fecha formateada según la configuración regional

Recomendaciones

  • Al trabajar con fechas, es importante tener en cuenta la zona horaria del usuario. Para obtener la fecha y hora en la zona horaria del usuario, podemos utilizar el método toDateString().
  • Es importante validar las entradas de usuario antes de crear objetos Date para evitar errores.

            const fechaHora = Temporal.PlainDateTime.from('2025-02-03T15:45');
            console.log(`Fecha y Hora: ${fechaHora.toString()}`); // "2025-02-03T15:45:00"
